# Cargo

A truck and cargo distribution game.

## Install


### Native

#### Prerequisite

* Linux or Macos. Other may work
* Git https://git-scm.org
* SBT https://github.com/sbt/sbt

#### Source

    git clone https://github.com/flurdy/cargo

#### Native Run

    cd cargo

    sbt run
